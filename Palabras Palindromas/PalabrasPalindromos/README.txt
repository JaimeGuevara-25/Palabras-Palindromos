NUMEROS PRIMOS

Requisitos: 
- Tener instalado la java virtual Machine o algún IDE.

Instrucciones para ejecutarlo:
Forma 1:
- Ubicar el archivo NumerosPrimos.JAR del proyecto en la siguiente ruta:
Ruta: "NumerosPrimos\dist"

-Cambia al directorio donde está el archivo y ejecútalo. 
Utilizar el comando "cd" para cambiar al directorio correcto. 
Una vez dentro del mismo, ejecuta 
"java -jar NumerosPrimos" 

Forma 2:
-Abrir la carpeta del proyecto entero con algún IDE y correrlo

Instrucciones para utilizarlo
:
1. Ingresa el número que deseas saber si es número primo o no
2. Presiona ENTER y dará respuesta
3. Seguidamente podrás ingresar algun otro para verificar, si quieres salir, ingresa cualquier letra y presiona ENTER.
